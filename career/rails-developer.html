Essentials

  • Good understanding of Ruby
  • Professional usage of Ruby on Rails and friends (Sinatra, etc.)

Bonus

  • Git
  • Experience with or interest in other backend technologies such as Elixir, Node.js, etc.
  • JS Frameworks such as Angular, Ember.js, React (we hear you, React is a library), etc.
  • Mobile development (ionic, React Native)
  • API development
  • Practicing Test Driven Development / BDD

We won’t bore you with “you must be a rockstar developer” – but do recommend that you are at least a medium level developer. We are not a standard employer, so if you want to chat with us, skype is but a call away.
