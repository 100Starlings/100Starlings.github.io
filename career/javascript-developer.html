Essentials

  • Good understanding of ECMAScript 6
  • Understanding and day-to-day usage of HTML 5 and related front-end skills

Bonus

  • Git
  • Frameworks such as Angular, Ember.js, React (we hear you, React is a library), etc.
  • Experience with or interest in backend technologies such as Ruby on Rails, Elixir, Node.js, etc.
  • Mobile development (ionic, React Native)

We won’t bore you with “you must be a rockstar developer” – but do recommend that you are at least a medium level developer. We are not a standard employer, so if you want to chat with us, skype is but a call away.
